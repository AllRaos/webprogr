Організація та налаштування мереж у Windows

Провідні мережі

Провідна мережа — це тип комп'ютерної мережі, в якій пристрої з'єднуються між собою за допомогою фізичних кабелів, таких як Ethernet-кабелі. Вона забезпечує стабільне та швидке з'єднання, що робить її ідеальною для передачі великих обсягів даних, роботи з високонавантаженими програмами або використання в умовах, де потрібна висока надійність. Провідні мережі зазвичай використовуються в офісах, будинках чи дата-центрах, де важлива безперебійна робота та мінімальні затримки. Основними компонентами такої мережі є маршрутизатори, комутатори, кабелі та мережеві адаптери, а налаштування в Windows дозволяє легко конфігурувати параметри підключення, такі як IP-адреса чи DNS.

IP-адреса (Internet Protocol Address) — це унікальний номер твого пристрою в мережі. Вона схожа на поштову адресу, тільки для комп’ютера. Приклад: 192.168.1.10

Маска підмережі (Subnet Mask) — це параметр, який говорить, яка частина IP-адреси належить до мережі, а яка — до конкретного пристрою. Найпоширеніша маска: 255.255.255.0

Шлюз (або основний шлюз, Default Gateway) — це точка виходу з локальної мережі в Інтернет. Зазвичай це адреса твого маршрутизатора (роутера), наприклад: 192.168.1.1

DNS (Domain Name System) — це сервіс, який перетворює назви сайтів на IP-адреси. Наприклад: коли ти вводиш google.com, DNS знаходить його IP-адресу (типу 142.250.185.206), і комп’ютер уже з нею працює.

Налаштування провідної мережі:

  1. Відкрити налаштування мережі:
    • Панель керування → Центр мережних підключень і спільного доступу
    • Або: Параметри → Мережа та Інтернет → Ethernet
  2. Перевірити стан з'єднання:
    • Клікнути на Ethernet → "Властивості"
    • У вкладці "Протокол Інтернету версії 4 (TCP/IPv4)" встановити:
      • Автоматичне отримання IP-адреси, або
      • Вручну ввести IP-адресу, маску підмережі, шлюз і DNS
  3. Перевірка доступу до Інтернету:
    • Використати команду ping у командному рядку: ping google.com

Безпровідні мережі

Безпровідна мережа — це тип комп'ютерної мережі, в якій пристрої з'єднуються між собою без використання фізичних кабелів, за допомогою радіохвиль, найчастіше через технологію Wi-Fi. Вона забезпечує мобільність і зручність, дозволяючи підключати до мережі смартфони, ноутбуки, планшети та інші пристрої в межах зони покриття. Безпровідні мережі широко використовуються в будинках, офісах, кафе та громадських місцях, де потрібен гнучкий доступ до інтернету. Основними компонентами є маршрутизатори або точки доступу, а налаштування в Windows дозволяє швидко підключитися до мережі, ввівши пароль, та конфігурувати параметри, такі як автоматичне підключення чи вибір частотного діапазону.

IP-адреса (Internet Protocol Address) — це унікальний номер твого пристрою в мережі. Вона схожа на поштову адресу, тільки для комп’ютера. Приклад: 192.168.1.10

Маска підмережі (Subnet Mask) — це параметр, який говорить, яка частина IP-адреси належить до мережі, а яка — до конкретного пристрою. Найпоширеніша маска: 255.255.255.0

Шлюз (або основний шлюз, Default Gateway) — це точка виходу з локальної мережі в Інтернет. Зазвичай це адреса твого маршрутизатора (роутера), наприклад: 192.168.1.1

DNS (Domain Name System) — це сервіс, який перетворює назви сайтів на IP-адреси. Наприклад: коли ти вводиш google.com, DNS знаходить його IP-адресу (типу 142.250.185.206), і комп’ютер уже з нею працює.

Налаштування роутера у безпровідній мережі

  1. Підключення до роутера:
    • Підключи роутер до електромережі.
    • Встав кабель інтернету (від провайдера) в порт WAN / Internet.
    • Підключи комп’ютер до роутера:
      • Через Ethernet-кабель, або
      • Через Wi-Fi (зазвичай на зворотньому боці роутера написано ім’я мережі та стандартний пароль).
  2. Вхід у налаштування роутера
    • Відкрий браузер.
    • Уведи одну з адрес в адресний рядок: 192.168.0.1 або 192.168.1.1
    • Введи логін і пароль для входу(Логін: admin; Пароль: admin або 1234):
  3. Налаштування інтернету (WAN)
    • Увійшовши, шукай вкладку Network / Інтернет / WAN і вибери тип підключення:
      • Dynamic IP (Динамічна IP) — якщо провайдер нічого особливого не вимагає.
      • PPPoE — якщо тобі дали логін і пароль для доступу (часто використовується).
      • Static IP — якщо провайдер дав тобі конкретні IP-дані.
  4. Налаштування Wi-Fi (Wireless)
    • Зайди у розділ Wireless / Wi-Fi Settings / Wireless Settings.
    • Налаштуй основні параметри:
      • Wireless Network Name (SSID): Назва твоєї Wi-Fi мережі.
      • Enable Wireless: Увімкнути Wi-Fi (поставити галочку)
      • Region: Обери свою країну
  5. Захист мережі (пароль)
    • Перейди до розділу Wireless Security / Security.
    • Вибери:
      • Security Mode: WPA2-PSK (найкраща безпека для домашнього інтернету)
      • Password / PSK Key: Введи свій пароль (мін. 8 символів)
    • Зберегти і перезагрузити роутер.

Налаштування безпровідної мережі:

  1. Підключення до мережі:
    • Клікнути на іконку Wi-Fi у панелі задач
    • Вибрати доступну мережу
    • Ввести пароль, якщо необхідно
  2. Налаштування адаптера:
    • Панель керування → Центр мережевих підключень → Зміна параметрів адаптера
    • Правий клік по "Бездротове мережеве з'єднання" → Властивості
    • Налаштування TCP/IPv4 автоматично або вручну
